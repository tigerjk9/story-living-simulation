# 이미지 저장 안내

업로드하신 "Story Living Simulation" 로고 이미지를 이 폴더에 저장해주세요.

## 저장 방법

1. 업로드하신 이미지를 다운로드하세요
2. 파일명을 `story-living-logo.png`로 변경하세요
3. 이 폴더(`public/`)에 저장하세요

## 파일 경로
```
public/story-living-logo.png
```

**중요**: Vite는 public 폴더의 파일을 루트 경로에서 제공합니다.
- 파일 위치: `public/story-living-logo.png`
- 접근 URL: `/story-living-logo.png` (public 제외)

이 이미지는 다음 용도로 사용됩니다:
- 웹사이트 파비콘 (브라우저 탭 아이콘)
- 소셜 미디어 미리보기 이미지 (Open Graph)
- Twitter 카드 이미지

## 권장 사항
- 파일 형식: PNG (투명 배경 권장)
- 최소 크기: 512x512px (정사각형 권장)
- 최대 파일 크기: 5MB 이하
